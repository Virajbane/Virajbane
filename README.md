# 🚀 Featured Projects:

### 🗄️ AI DB Agent — Universal Database Natural Language Interface
Full-stack AI-powered NL interface for MongoDB, PostgreSQL, MySQL, Redis & Supabase using Next.js, FastAPI, and Ollama (Qwen2.5-Coder 7B). Supports multilingual queries (English, Hindi, Marathi) with hybrid intent classification and schema-aware, SQL-injection-safe query generation.

### 🎓 EDU.AI — AI-Powered Classroom Learning Platform
Built at Mumbai Hack 2024 — features lecture summarization, transcript analysis, and keyframe extraction via Python ML microservices, integrated with Next.js, Tailwind, Supabase, Prisma, and Google OAuth. Delivers personalized study routines and real-time learning analytics.

### 🧠 Resume–Job Matcher
Fine-tuned BERT (110M params) on resume–job description pairs for semantic similarity and job-fit scoring, using a `Resume [SEP] Job Description` classification pipeline with softmax-based probability scoring — going beyond keyword matching.

### 🚖 Move-On — Ride-Hailing Web Application
Full-stack ride-hailing platform with Next.js, Tailwind CSS, Clerk Authentication, Google Maps API, and Razorpay, featuring real-time ride tracking, route visualization, and secure payments. Deployed on Netlify.
